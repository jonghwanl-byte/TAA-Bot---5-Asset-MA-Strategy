# Tactical Asset Allocation (TAA) Bot - 5-Asset MA Strategy

본 프로젝트는 5개 금융 투자 자산(한국/중국/인도 주식 3종 + 한국 채권 2종)에 이동평균선(MA) 기반의 전술적 비중 조절 전략을 적용하고, 그 결과를 텔레그램으로 자동 보고하는 Python 패키지입니다.

---

## ⚙️ 전략 설정

* **자산 (Tickers):**
    * `102110.KS` (TIGER 200)
    * `283580.KS` (TIGER 200 중공업) - *참고: 이 티커는 2017년 상장입니다.*
    * `453810.KS` (TIGER 인도니프티50)
    * `148070.KS` (KOSEF 국고채 10년)
    * `385560.KS` (KODEX 200 미국채 혼합)

* **기본 비중 (Base Weight):** 모든 자산 20% (균등 배분)

* **MA 지표:** 20일, 120일, 200일 이동평균선

* **비중 조절 (시나리오 A):**
    3-MA 신호 개수에 따라 100%, 75%, 50%, 0%로 투자 비중 변경. (단순 가격 > MA 기준)

| MA 신호 | 실제 투자 비중 |
| :--- | :--- |
| 3개 선 **모두 위** | **100%** (예: 20% 전액 투자) |
| 2개 선 **위에** | **75%** (예: 15% 투자) |
| 1개 선 **위에** | **50%** (예: 10% 투자) |
| 3개 선 **모두 아래** | **0%** (투자 안함 / 현금) |
